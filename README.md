#  Earthquake Prediction using Machine Learning

##  Project Overview
This project predicts earthquake magnitude and depth using machine learning.

##  Model Used
- Neural Network (Dense Layers)
- Activation: ReLU
- Optimizer: Adam

##  Features
- Timestamp
- Latitude
- Longitude

##  How to Run

```bash
pip install -r requirements.txt
python earthquake.py
